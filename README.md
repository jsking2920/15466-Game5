# Skull Shooter

Author: Scott King, William Ozeas <br>
Design: Skull shooter is about shooting skulls! And not dying! So shoot skulls and don't die!

Screen Shot:

![Screen Shot](screenshot.png)

How To Play:
- Move: wasd
- Shoot: Left mouse button
- Reload: r
